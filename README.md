# PlatePilot

## Overview
PlatePilot is a smart student meal planning app that helps users eat healthier and save money. It recommends meals based on time, cost, and nutritional value, allowing students to schedule weekly meals, build grocery lists, and track spending + calories.

## Product Spec

### 1. User Stories
- User can browse meals in a recipe feed
- User can tap a meal to see details
- User can save meals to favorites
- User can add meals to a weekly planner
- User can view generated grocery list
- User can view weekly budget & calorie summary

### 2. Screens
- Home / Meal Feed
- Meal Detail Screen
- Weekly Meal Plan Screen
- Grocery List Screen
- Budget & Nutrition Dashboard
- Favorites Screen

### 3. Navigation
**Tab Bar**
- Home
- Planner
- Grocery
- Profile

### 4. Data Persistence
- UserDefaults or CoreData for favorites, planner, and grocery items

### 5. Networking
Using TheMealDB API
`https://www.themealdb.com/api/json/v1/1/search.php?s=`
